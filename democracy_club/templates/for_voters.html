I want to help voters find their voting information…

Election Widget

You can embed our 'Widget' for free on on any web page. The embedding process is as simple as copying and pasting some code, and does not require a web developer. We have two widgets to choose from:

  • Combined elections widget. This shows elections, candidates and polling stations to the user.
  • Polling station widget. This displays only the location of the user's polling place.

If you would like a custom election widget to fit with your website's theme, please get in touch to discuss this with us.

Get the Data

Democracy Club offers its elections data as CSV downloads or in API format.

Researchers and journalists can find candidates and election results data on our candidate database.Our CSVs are free to download and use with credit.

Web Developers can access our elections data via an API, allowing you to build your own voter information tools. Please see our API pages for more information.
